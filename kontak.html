Pertanyaan yang Sering Diajukan

Temukan jawaban atas pertanyaan umum seputar program perusahaan kami

Apa saja program unggulan yang ditawarkan perusahaan?

Kami menawarkan beberapa program unggulan untuk pengembangan bisnis:

  • Program Strategi Bisnis: Konsultasi pengembangan strategi jangka panjang
  • Transformasi Digital: Solusi untuk digitalisasi operasional bisnis
  • Pengembangan SDM: Pelatihan kepemimpinan dan pengembangan kompetensi
  • Efisiensi Operasional: Program peningkatan produktivitas dan efisiensi

Bagaimana proses pendaftaran untuk mengikuti program?

Proses pendaftaran dapat dilakukan melalui beberapa langkah berikut:

  1. Kunjungi halaman pendaftaran program di website kami
  2. Isi formulir pendaftaran dengan data perusahaan yang valid
  3. Pilih program yang diinginkan dan jadwal pelaksanaan
  4. Tim kami akan menghubungi Anda dalam 2 hari kerja untuk verifikasi
  5. Setelah verifikasi, Anda akan menerima email konfirmasi dan instruksi pembayaran
  6. Program akan diaktifkan setelah pembayaran diverifikasi

Apakah ada persyaratan khusus untuk mengikuti program?

Persyaratan untuk mengikuti program bervariasi tergantung jenis programnya:

Untuk semua program:

  • Perusahaan terdaftar secara legal
  • Memiliki minimal 10 karyawan
  • Bersedia berkomitmen untuk implementasi program

Program khusus:

  • Program Transformasi Digital: memiliki infrastruktur IT dasar
  • Program Pengembangan SDM: menyediakan fasilitas pelatihan
  • Program Efisiensi Operasional: memiliki data operasional 6 bulan terakhir

Berapa lama durasi program dan bagaimana jadwal pelaksanaannya?

Durasi program bervariasi:

  • Program Intensif: 2-4 minggu (full-time, Senin-Jumat)
  • Program Reguler: 3-6 bulan (2-3 sesi per minggu)
  • Program Eksekutif: 1 tahun (1 sesi per minggu + konsultasi bulanan)

Jadwal pelaksanaan dapat disesuaikan dengan kebutuhan perusahaan peserta. Umumnya sesi dilaksanakan pada:

  • Hari kerja: 09.00-12.00 atau 13.00-16.00 WIB
  • Sabtu: 09.00-15.00 WIB (untuk program tertentu)

Apakah program ini menyediakan sertifikat kelulusan?

Ya, semua program kami menyediakan sertifikat kelulusan dengan ketentuan:

  • Peserta menyelesaikan minimal 80% sesi program
  • Menyelesaikan semua tugas dan proyek yang diberikan
  • Mencapai nilai minimal kelulusan pada asesmen akhir

Sertifikat yang diberikan berupa:

  • Sertifikat Penyelesaian Program (untuk semua peserta yang memenuhi syarat)
  • Sertifikat Prestasi (untuk peserta dengan nilai tertinggi)
  • Sertifikat Implementasi (setelah pelaksanaan proyek di perusahaan)

Bagaimana metode pembayaran untuk program ini?

Kami menyediakan beberapa opsi pembayaran yang fleksibel:

  • Pembayaran Penuh: Diskon 10% untuk pembayaran lunas sebelum program dimulai
  • Cicilan: 50% di awal, 50% di tengah program
  • Termin: 30% saat pendaftaran, 40% di pertengahan, 30% di akhir program

Metode pembayaran yang diterima:

  • Transfer Bank (BCA, Mandiri, BNI, BRI)
  • Kartu Kredit (Visa, MasterCard)
  • Virtual Account
  • E-Wallet (OVO, GoPay, DANA)

Untuk perusahaan dengan peserta lebih dari 5 orang, tersedia paket khusus dengan diskon tambahan.
